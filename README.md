## Warehouse-control-API

### Description:
App for warehouse control by API.

### get started:
git clone project:
https://github.com/AxmetES/Warehouse-control-API.git

### got to project rep:
```cd /Warehouse-API```

### start app:
```docker-compose up --build```

### run tests:
```docker-compose run tests```

### Docs:
Swagger openapi.json

http://localhost:8000/docs#/
